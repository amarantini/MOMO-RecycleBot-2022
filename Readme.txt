Game Rule:
1. Earn 500 to win at Level 1. Earn 1000 to win at Level 2
2. Supervisor will conduct sanitory check every 100s. It will issue warning if there are too much trash in a region. 3 warnings cause game over. Robot is fired.
3. There are 3 types of trash:
	- Recyclable trash that by collection can earn coins
	- Trash that need wiping
	- Normal trash
4. Player need to send recyclable trash to recyclable trash processor. Use left and right arrow to control direction of conveyor belt
5. Tools can be bought at store to help collect trash

Cheat Code:
Press Q to add 100 coins
